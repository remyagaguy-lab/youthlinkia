# Agent Rules for YouthLinkia

Throughout the development of the YouthLinkia project, you MUST actively apply the following installed skills whenever relevant:

- **frontend-design** & **ui-ux-pro-max**: Use these for all UI/UX development. Ensure distinctive, premium aesthetics and modern, responsive layouts. Do not settle for basic or generic designs.
- **react-best-practices** & **composition-patterns**: Apply these to all React and Next.js development. Focus on performance, modularity, compound components, and scalability.
- **bencium-innovative-ux-designer** & **bencium-controlled-ux-designer**: Use these depending on the context—innovative/experimental designs for marketing/public pages, and controlled/systematic patterns for dashboards or admin interfaces.
- **web-design-guidelines**: Strictly adhere to established web interface principles during all UI reviews and implementation.
- **accesslint (audit, diff, scan)**: Proactively evaluate and fix WCAG 2.2 accessibility issues to ensure the platform is inclusive.
- **react-native-skills**: Apply to any mobile-specific or React Native codebase segments.

Always refer to these skills when planning architecture, writing code, or reviewing the user interface.
